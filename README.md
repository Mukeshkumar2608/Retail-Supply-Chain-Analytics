📦 Retail Supply Chain Analytics | Excel + MySQL + Power BI

## 📌 Project Overview

This project analyzes a **Global Retail Supply Chain dataset** using **Excel, MySQL, and Power BI** to generate meaningful business insights and support data-driven decision-making.

The project follows the complete **Data Analytics Lifecycle**, including data cleaning, SQL analysis, and interactive dashboard creation.

---

## 🔄 Project Workflow

### **1. Data Cleaning (Microsoft Excel)**

* Removed duplicate records
* Handled missing values
* Corrected inconsistent data
* Standardized date formats
* Verified data types
* Prepared the dataset for SQL analysis
* Exported the cleaned dataset as **CSV (UTF-8)**

### **2. Data Analysis (MySQL)**

* Imported the cleaned CSV into MySQL
* Wrote **50+ business-focused SQL queries**
* Performed sales, profit, customer, product, regional, and supply chain analysis
* Used SQL functions such as:

  * Aggregate Functions
  * GROUP BY
  * ORDER BY
  * HAVING
  * CASE Statements
  * Date Functions
  * Joins (where applicable)
  * Subqueries

### **3. Data Visualization (Power BI)**

Developed an interactive dashboard containing:

* 💰 Total Sales
* 📈 Total Profit
* 📦 Total Orders
* 👥 Total Customers
* 📊 Profit Margin
* 💳 Average Order Value
* 📅 Monthly Sales Trend
* 🌍 Sales by Region
* 🏷️ Category Sales Analysis
* 👤 Customer Segment Analysis
* ⭐ Top Selling Products
* 🎯 Top Customers

Interactive slicers:

* Year
* Region
* Segment
* Category

## 📊 Key Business Insights

* Technology category generated the highest sales.
* Consumer segment contributed the highest revenue.
* High-discount products often resulted in lower profit margins.
* Regional sales performance varied significantly across markets.
* Monthly sales trends highlighted seasonal demand.
* Product performance analysis identified top-selling and low-performing products.
